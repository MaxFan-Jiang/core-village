# 把你的村民放進遊戲 🎮

歡迎來到芯覺村。這份指南帶你完成**人生第一個 PR**：放進一個專屬你的村民角色。

全程在瀏覽器點一點就好，**不用裝任何軟體、不用碰終端機、不用會寫程式**。大概 5 分鐘。

> 🖼️ 看圖跟著做更快：打開 [`onboarding.html`](onboarding.html)（圖文＋動畫的保母教學）。
> 這份文字版是它的對照表。

---

## 你要做的事，一句話

複製一個範本檔 → 改成你的角色 → 送出（PR）。引擎會自動把你排上場，幾分鐘後你就出現在線上遊戲裡。

---

## 💡 你的村民，是「平行時空的你」

村裡那 12 個預設村民，是這一班真實的共學夥伴。你要 PR 的那一張，可以是**另一個版本的你**：

- 如果當初選了另一條路（理組／文組、留下／離開），那個你會是什麼角色？
- 你的夢想如果**已經實現了**，那個你會不會更強？（睡飽的人攻擊力是不是更高？）
- 你最想成為的樣子，做成一張卡。

地板很低（填幾個格子就好），天花板很高（想設計角色故事、加入條件，甚至自己的玩法都可以）。先把第一張卡 PR 進來，再慢慢長。

## 一步一步（瀏覽器版，零基礎）

### 1. Fork（複製一份到你的帳號）
在這個 repo 右上角按 **Fork**。GitHub 會幫你複製一份到你自己的帳號底下。

### 2. 進 `villagers/` 資料夾
在你 fork 的那一份裡，點進 `villagers/` 資料夾。裡面每一個 `.json` 就是一位村民。

### 3. 開新檔案，貼上範本
- 先點開 [`villagers/_example.json`](villagers/_example.json)，把它的內容整段複製起來。
- 回到 `villagers/`，按 **Add file → Create new file**。
- 檔名打 `villagers/你的代號.json`（例如 `villagers/maxfan.json`，**只能用小寫英文、數字、減號**）。
- 把剛剛複製的內容貼進去。

### 4. 改成你的角色
照下面說明改每一個欄位（改完記得 `id` 要和檔名一樣）：

```json
{
  "id": "你的代號",          // 和檔名一樣，小寫英文／數字／減號，獨一無二
  "name": "顯示名字",         // 卡片上顯示的名字，自己取，不必真名
  "job": "你的本行 · 一句話",  // 例：插畫漫畫 · 視覺創作
  "emoji": "🌟",             // 一個 emoji，當你的招募卡頭像
  "hp": 100,                 // 血量 50–220
  "atk": 18,                 // 攻擊 8–40
  "spd": 11,                 // 速度 5–20（決定出手順序）
  "skill": "shield",         // 從下面技能表挑一個
  "cry": "上場前想說的一句話。", // 戰吼／台詞
  "story": "兩三句你的角色故事。", // 選用：你的故事（平行時空／夢想實現版的你）
  "join": "一個有梗的加入條件。"   // 選用：別人要把你加進隊伍的條件
}
```

> `story` 和 `join` 是**選用**的，不填也沒關係。填了，別人在村莊裡走到你面前就會看到，這是讓你的角色活起來的地方。

### 5. 送出（Pull Request）
- 按綠色 **Commit changes**（存進你的 fork）。
- 回到原本的 repo，按 **Pull requests → New pull request → Create pull request**。
- 標題寫「加入我的村民：你的代號」就好，送出。

### 6. 等綠燈，然後上線 ✅
- 一個機器人（CI）會自動檢查你的卡。**有錯它會用中文告訴你哪裡要改**，改好再 push 就會重檢。
- 綠燈後，維護者會把你 merge 進來。**幾分鐘後**，你的村民就自動出現在線上遊戲的招募大廳了。

## 技能表（`skill` 只能填這 12 個之一）

| 填這個 | 技能 | 效果 |
|---|---|---|
| `shield` | 風險對沖 🛡️ | 全隊獲得護盾，吸收下一次傷害 |
| `taunt` | 挺身 🦺 | 嘲諷所有敵人吸引火力到自己，獲得大護盾並大幅減傷（坦克） |
| `freeze` | 罰站 ❄️ | 凍結一個敵人，讓牠罰站 2 回合 |
| `assassinate` | 自動化連擊 🗡️ | 連擊 3 個敵人、高爆擊；爆擊會疊「殺意」滾雪球 |
| `dodge` | 分鏡預判 🎯 | 閃避敵人的下一次攻擊 |
| `heal` | 散熱模組 💧 | 治療血量最低的隊友 |
| `summon` | 檔期促銷 🎁 | 召喚一個援軍小幫手 |
| `corrode` | 蝕刻 ☠️ | 攻擊，並讓全體敵人「中毒」＋「易傷」，可疊層（群傷） |
| `empower` | 市場賦能 📈 | 全隊攻擊力與爆擊率提升 |
| `hypnosis` | 全場催眠 🌀 | 凍結所有敵人 1 回合，並使其「易傷」 |
| `overclock` | 超頻 ⚡ | 對主目標 2.3 倍爆發、餘波炸傷全場；下回合過熱（群傷） |
| `regen` | 節能續航 ➕ | 全隊獲得「再生」，3 回合內每回合回血 |
| `armorbreak` | 精密拆解 🔩 | 重擊一個敵人；殘血直接拆解（即死，魔王除外），否則疊 2 層「易傷」 |

> 💡 **挑技能的小訣竅**：把你的本行翻成一個技能。做業務的選賦能、當老師的選罰站、寫程式的選連擊、做風控的選護盾。

---

## 平衡規則（機器人會幫你擋）

- 血量 `hp` 50–220、攻擊 `atk` 8–40、速度 `spd` 5–20。填太誇張（例如 `hp: 99999`）會被退回。
- 想當爆擊流？可以多加一個 `"crit": 0.42`（爆擊率，0–0.6），搭配 `assassinate` 滾殺意。
- 名字／台詞不能放 `<` `>` 這種符號（資安考量）。

## 進階：叫 Claude Code 幫你做

如果你在用 Claude Code，直接跟它說：

> 「幫我在 `villagers/` 做一張我的村民卡，我的本行是 ＿＿＿，幫我挑一個適合的技能，然後開一個 PR。」

它會照這份 schema 生檔、檢查、開 PR。

> 🛠️ 自寫技能（`skill: "custom"` ＋ 自己的技能碼）目前還沒開放——在公開遊戲裡跑別人 PR 進來的程式碼有資安風險，要走人工 code review 直接併進引擎。想做的話在 PR 裡提，我們一起看。
